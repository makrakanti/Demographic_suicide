The .xls file is the output from the discussed model of adult survival and capture probability fit using E-Surge. Several sheets are important: "Model definition" contains the matrices and the corresponding phrases used in the GEMACO interface of the software; "Deviance" provides the QAIC, deviance, etc.; "Reduced set of parameters" gives the parameter estimates - the meaning of the codes for the different variables are provided in parentheses at first mention.
